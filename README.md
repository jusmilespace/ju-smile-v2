# 🌿 Ju Smile 減脂日誌 v2.1

**Ju Smile 減脂日誌** 是一款支援離線使用（PWA）的個人化健康日誌 App。  
幫助你追蹤每日飲食、運動、喝水量與體重變化，  
未來也將支援雲端同步與週/月報表分析。

🌐 **線上版本：**  
👉 [https://jusmilespace.github.io/ju-smile-v2/](https://jusmilespace.github.io/ju-smile-v2/)

---

## 🧩 主要功能

| 類別 | 功能 |
|------|------|
| 📅 **日期切換** | 選擇、快速切換日期，資料自動分天保存。 |
| 🧍‍♀️ **體重追蹤** | 每日輸入體重，報表頁可查看 7/30/90 天趨勢。 |
| 💧 **喝水紀錄** | 可快速 +200ml / +500ml 或自訂輸入。 |
| 🍱 **飲食紀錄** | 食物名稱、份量、熱量與蛋白質皆可輸入，會自動計算總攝取。 |
| 🏃‍♀️ **運動紀錄** | 可輸入運動名稱、時長、熱量與備註，自動彙總消耗熱量。 |
| 📊 **報表分頁** | 顯示體重趨勢折線圖（支援離線），可切換天數範圍。 |
| 🔁 **跨裝置同步說明** | 提供匯出/匯入 JSON 檔手動同步教學。 |
| ⚙️ **PWA 離線支援** | 可「加入主畫面」後離線使用。 |

---

## 📲 使用方式

1. 開啟網頁 [Ju Smile 減脂日誌](https://jusmilespace.github.io/ju-smile-v2/)  
2. 第一次載入後可選擇「加入主畫面」→ 成為獨立 App 使用。  
3. 點選日期 → 輸入體重、飲食、喝水量、運動資料。  
4. 點右上角「匯出」可下載 JSON 備
